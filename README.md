# Hospital Management System

A simple Python project to manage patients and appointments.

## Features

- Add and list patients
- Book and view appointments
- Store data in a JSON file

## Setup

```bash
pip install -r requirements.txt
python main.py
```

## Project Structure

```
hospital-management/
├── main.py           # Run the app
├── patient.py        # Patient model
├── appointment.py    # Appointment handling
├── data/
│   └── patients.json # Saved data (created on first run)
└── requirements.txt
```
